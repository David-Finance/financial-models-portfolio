# Financial Models Portfolio

This repository contains Excel-based financial models developed for financial analysis and decision-making.

## Projects

### 1. Portfolio Risk & Return Optimization Model
- Built using historical stock data
- Includes covariance matrix and portfolio risk calculation
- Optimized using Excel Solver
- Simulates different risk-return scenarios

### 2. Payroll Financial Simulation Model
- Automated salary, tax, and deduction calculations
- Uses complex Excel formulas (IF, VLOOKUP, IFERROR)
- Integrates ISR and IMSS tables
- Designed for efficient and accurate financial outputs
